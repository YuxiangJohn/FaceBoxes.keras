### Overview

This is a Keras port of FaceBoxes: A CPU Real-time Face Detector with High Accuracy, introduced by Shifeng Zhang et al. in the paper [FaceBoxes: A CPU Real-time Face Detector with High Accuracy](https://arxiv.org/pdf/1708.05234.pdf).
